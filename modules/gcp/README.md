# GCP Modules

Terraform modules for deploying Corelight infrastructure on Google Cloud Platform.

| Module | Purpose |
|---|---|
| [`sensor`](./sensor/) | Managed Instance Group (MIG) sensor deployment |
| [`enrichment`](./enrichment/) | Enrichment service (Cloud Run + Pub/Sub + Cloud Scheduler) |

For deployment guidance and architecture details, see the official Corelight documentation.
